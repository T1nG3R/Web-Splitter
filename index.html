    📖 How to use & merge chunks

    Splitting a File

    1. Drag & drop or click the zone to select a file.
    2. Specify your desired chunk size.
    3. Choose the naming convention.
    4. Click Split File. The chunks should start downloading automatically.
    5. If necessary, allow this page to download multiple files.
    6. If the download hasn’t started, download the chunks manually.

    Rejoining / Merging Chunks

    Chunk files are generated using standard formats. You can rejoin them easily:

    Method 1: Using 7-Zip or another file archiver (7-Zip style only)

    If you used the default 7-Zip style naming, you can right-click the first file (ending in .001) and extract it using your file archiver to reconstruct the original file. You don't need to extract any other parts as they are combined automatically. (Note: Part style naming is not supported by 7-Zip; please use the command-line method below instead).

    Method 2: Command Line (Fastest)

    Open your terminal, run the command below to join the chunks. Click "Copy" to copy it.

    Windows Command Prompt 
    copy /b "archive.zip.00*" "archive.zip"
    Windows PowerShell 
    cmd /c 'copy /b "archive.zip.00*" "archive.zip"'
    macOS / Linux Terminal 
    cat "archive.zip.00"* > "archive.zip"

    All processing happens locally in your browser. Your files are never uploaded.
